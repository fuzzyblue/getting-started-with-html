Web basics

Servers: Where the information on the web lives.

Web Browsers: Programs that convert HTML into human-ready information.

HTTP: HyperText Transfer Protocol, is the rules by which information is transferred over the web, and how a Web Browser retrieves information from a Server.

HTML: HyperText Mark-up Language, which determines how information is displayed in a Web Browser. HTML documents contain instructions on both the text to be displayed, as well as how the text is actually shown.

HTML Element: The basic unit of a HTML document. A HTML Element may be either inline (ie, contiguous) or block (ie, discrete).

    Format: <tag>content</tag>

HTML Tags: A HTML tag marks the start & end of a HTML Element.

    Format: <tag>element contained within the tags</tag>

HTML Attribute: A HTML attribute defines a property of a HTML Element, and is contained within the start of the HTML Tag.

    Format: <tag attribute>element contained within the tags</tag>

    Example: <a href="www.link.com/specificlink">element contained within the tags</a>

Contrary to the Stage 0 lesson & commonly accepted wisdom, computers are NOT stupid. A computer is more accurately 'literal' in how it interprets human instructions. 'Stupid' implies a capacity for intelligence which computers — at present — do not possess. :-)
